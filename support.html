Quick troubleshooting

  • If the toolbar does not appear, refresh the page and click the extension icon again.
  • Some Chrome internal pages do not allow extensions. Try a normal webpage or PDF.
  • If saving does not include the page, make sure the page is fully loaded before pressing save.
  • If highlighting behaves strangely, refresh the page and try highlighting a smaller text selection.
