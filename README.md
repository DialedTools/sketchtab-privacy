# SketchTab Website With Support Form

This version includes:
- index.html = Homepage
- support.html = Support page with email support form
- privacy.html = Privacy policy
- thanks.html = Confirmation page after form submit
- README.md = This guide

The support form sends submissions to:
eduriai0515@gmail.com

Important:
The form uses FormSubmit, a static-form email service. The first submission may send a confirmation email to eduriai0515@gmail.com. Open that email and confirm the form before expecting normal support messages.

Upload/update on GitHub:
1. Open your GitHub repository: sketchtab-privacy
2. Click "Add file" > "Upload files"
3. Upload all files from this ZIP:
   - index.html
   - support.html
   - privacy.html
   - thanks.html
   - README.md
4. If GitHub asks, choose to replace existing files.
5. Click "Commit changes".
6. Wait 1-3 minutes for GitHub Pages to update.

Your URLs should be:

Homepage URL:
https://dialedtools.github.io/sketchtab-privacy/

Support URL:
https://dialedtools.github.io/sketchtab-privacy/support.html

Privacy policy URL:
https://dialedtools.github.io/sketchtab-privacy/privacy.html

Test the support form:
1. Open https://dialedtools.github.io/sketchtab-privacy/support.html
2. Submit a test message.
3. Check eduriai0515@gmail.com.
4. If FormSubmit sends a confirmation email, click the confirmation link.
5. Submit one more test message.
