# SketchTab Website Files

Upload these files to a GitHub repository and turn on GitHub Pages.

Files:
- index.html = Homepage URL
- support.html = Support URL
- privacy.html = Privacy policy URL
- README.md = This guide

Recommended repository name:
sketchtab-site

After GitHub Pages is enabled, your URLs will look like this:

Homepage URL:
https://YOUR-GITHUB-USERNAME.github.io/sketchtab-site/

Support URL:
https://YOUR-GITHUB-USERNAME.github.io/sketchtab-site/support.html

Privacy policy URL:
https://YOUR-GITHUB-USERNAME.github.io/sketchtab-site/privacy.html

For your GitHub account shown earlier, it will probably be:

Homepage URL:
https://dialedtools.github.io/sketchtab-site/

Support URL:
https://dialedtools.github.io/sketchtab-site/support.html

Privacy policy URL:
https://dialedtools.github.io/sketchtab-site/privacy.html

GitHub Pages steps:
1. Go to GitHub.
2. Create a new public repository named: sketchtab-site
3. Click "uploading an existing file".
4. Upload index.html, support.html, privacy.html, and README.md.
5. Click "Commit changes".
6. Go to Settings > Pages.
7. Source: Deploy from a branch.
8. Branch: main.
9. Folder: / root.
10. Save.
11. Wait 1-3 minutes and open the URLs.
